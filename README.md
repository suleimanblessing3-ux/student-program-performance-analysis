# student-program-performance-analysis
Data analysis project examining student performance metrics to optimize program outcomes using Python, SQL, and Power BI
📊 Project Overview
This project analyzes student performance data from educational technology programs to identify key factors that drive student success, optimize tutor allocation, and improve overall program outcomes. Using Python for data analysis and Power BI for visualization, the analysis provides actionable insights for program managers.
🎯 Business Problem
As a Programs Manager overseeing 70+ students per cohort across multiple training programs, understanding what drives student satisfaction and completion rates is critical. This analysis helps:

Identify at-risk students early for intervention
Optimize resource allocation and tutor assignments
Improve program design based on data-driven insights
Increase satisfaction rates from 90% to 95%
Key Questions Answered

Which factors most strongly predict student completion rates?
How does tutor assignment impact student satisfaction scores?
What is the optimal cohort size for maximum engagement?
Are there seasonal patterns in program performance?
Which programs have the highest ROI in terms of student outcomes?

Key Findings
🎓 Main Insights

Attendance is Critical: Students with 80%+ attendance are 3.5x more likely to complete programs
Optimal Cohort Size: 55-65 students achieve the best balance (82% completion, 4.3/5 satisfaction)
Tutor Quality Matters: Top-performing tutors achieve 88% completion vs 68% for lower-rated tutors
Seasonal Patterns: Q1 cohorts show 15% higher completion rates than other quarters
Technical Support: 3+ support tickets correlate with 25% lower satisfaction scores
